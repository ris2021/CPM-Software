Классический сапёр для КРИСС CP/M

http://criss.radio.ru

Команды управления:
Пробел - открытие клетки
Ввод - поставить/снять метку мины
Ctrl-Пробел - открытие всех соседних клеток

Осторожно: при открытии соседей никаких проверок нет!
Классической правило "первый ход мимо мины" не реализовано, удачи!

Minesweeper game for CRISS CP/M computer

http://criss.fun

Controls:
Space - open cell
Enter - mark bomb
Ctrl-Space - open all neighbours

Please report all bugs to mail@ogss.ru
